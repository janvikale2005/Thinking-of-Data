
# Data Thinking - List Operations

This repository contains Python programs that demonstrate basic list operations used in data thinking.

## Files Included
1. list_creation.py – Shows how to create lists in Python.
2. list_size.py – Finds the size of a list using the len() function.
3. list_shape.py – Demonstrates the shape of nested lists (rows × columns).
4. list_dimension.py – Explains 1D and 2D list structures.

## Objective
To understand list creation, size calculation, shape detection, and dimensions of lists using Python.

## Technology
Python
