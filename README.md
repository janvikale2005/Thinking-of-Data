# Data Thinking - List Operations (Jupyter Notebook)

This repository contains Jupyter Notebook programs demonstrating basic list operations used in data thinking and data analysis.

## Files Included
1. list_creation.ipynb – Demonstrates how to create lists in Python.
2. list_size.ipynb – Shows how to calculate the size of a list using the len() function.
3. list_shape.ipynb – Explains the shape of nested lists (rows × columns).
4. list_dimension.ipynb – Demonstrates 1D and 2D list structures.

## Objective
To understand list creation, size calculation, shape detection, and dimensions of lists using Python.

## Technology Used
- Python
- Jupyter Notebook
- Basic Data Structures
